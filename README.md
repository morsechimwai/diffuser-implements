### Examples of Usage

#### Example 1: Create a basic image

```bash
uv run python sdxl_txt2img.py \
  --prompt "a serene Japanese garden with cherry blossoms" \
  --output "garden"
```

#### Example 2: Create a high-quality image with adjusted parameters

```bash
uv run python sdxl_txt2img.py \
  --prompt "futuristic cityscape at night, neon lights, cyberpunk style" \
  --output "cyberpunk_city" \
  --steps 30 \
  --guidance_scale 7.5 \
  --width 1024 \
  --height 1024 \
  --seed 12345
```

#### Example 3: Create a artistic image

```bash
uv run python sdxl_txt2img.py \
  --prompt "oil painting of a cat wearing a crown, royal, majestic" \
  --output "royal_cat" \
  --steps 25 \
  --width 1024 \
  --height 768
```
